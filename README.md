# Lister dossiers & fichiers

Dir&Lister est un logiciel de bureau gratuit qui liste les dossiers et/ou fichiers de répertoires choisis. 

## Sélection simple & efficace
Glisser et déposer les dossiers à lister dans l’application à partir de l'Explorateur Windows.

L'ajout de répertoires peut aussi s'effectuer en téléchargeant une liste de chemins, sauvegardée dans un fichier texte. Ceci est efficace lorsque les mêmes chemins sont utilisés à plusieurs reprises.

## Formats de Sauvegarde
* csv
* txt
* presse-papier

Lorsque la recherche a été sauvegardée, le bouton "copier" permet de copier le résultat dans le presse-papier, peu importe l’option de sauvegarde sélectionnée au préalable.

## Options de recherche
* Nom de fichier et/ou dossier
* Chemin de fichier et/ou dossier
* Taille
* Date de création
* Extension de fichier
* Nombre de fichiers dans un dossier*
* Profondeur de la recherche: dossier principal, sous-dossier (1er niveau), récursivement
* Inclure ou exclure le dossier principal du résultat de recherche

*Lorsque la recherche récursive est sélectionnée, le nombre de fichiers par dossier correspond au nombre de fichiers directement sous le dossier. Sinon, il s’agit du nombre total des fichiers, incluant ceux des sous-dossiers.

## Options d’affichage
* Séparateur de données:
> * Point-virgule
> * Tabulation
> * Espace
> * Barre verticale
* Séparer le nom de fichier et dossier du chemin
* Séparer l’extension du fichier
* Afficher l’entête des colonnes

## Options de tri
* Trier par:
> * Chemin absolu
> * Chemin
> * Nom de fichier et/ou dossier
> * Taille
> * Date de création
> * Extension
* Tri ascendant ou descendant
* Ordre naturel de Windows ou non
